Sistema operativo:
Es el software principal que administra los recursos del computador. Permite ejecutar programas, manejar archivos y controlar el hardware.

Node.js:
Es un entorno de ejecución para JavaScript fuera del navegador. Permite crear servidores y aplicaciones backend de forma eficiente.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y la forma en que el servidor responde a las solicitudes de los usuarios.

Puerto:
Es un punto de comunicación que permite recibir y enviar datos. En esta actividad se utiliza el puerto 3000 para acceder al servidor.

Logs:
Son registros que almacenan información sobre la ejecución del sistema. Ayudan a monitorear actividades, detectar errores y realizar mantenimiento.
